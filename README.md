# LAUpdater
Linux Apps Updater

# Anleitung
Zum Start die .sh Datei starten und das root Passwort in das Terminal eingeben.

# Kompatibilität
Es funktioniert auf Linux-Systemen, auf denen das Paket apt installiert ist.
